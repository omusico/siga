Построение буферов

Этот инструмент создает новый полигональный слой с полигональными буферами для объектов исходного слоя.

Объекты исходного слоя могут быть любого типа (точка, линия, полигон). Для каждого объекта создается один или несколько (на равном расстоянии) полигональных кольцевых буферов. Также, если исходный слой полигональный, буферы могут быть внешними, внутренними или обоих типов.
