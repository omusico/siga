Слияние (Dissolve)

Этот инструмент выполняется с полигональным слоем. Полигоны с одинаковыми атрибутами будут объединены в один. Можно указать пространственный критерий "Выполнить слияние только для смежных объектов". Тогда объединение (слияние) будет выполнено только для рядом расположенных полигонов.
