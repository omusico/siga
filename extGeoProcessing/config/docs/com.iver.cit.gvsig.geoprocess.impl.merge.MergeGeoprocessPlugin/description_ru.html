Объединение слоев (Merge)

Этот инструмент работает с множеством входных наборов данных. Выходной набор данных будет содержать все объекты всех входящих наборов данных. Так как входные наборы данных могут иметь разную схему атрибутов, необходимо выбрать слой, на основе которого будет создана схема для выходного набора данных.
