Построение полигонов

Эта операция строит полигональный слой из тех объектов линейного слоя, которые образуют замкнутую область.

Все те линии, которые не входят в области, опционально могут быть добавлены во вспомогательные слои (висящие и вырезанные линии).

Опционально вы можете указать допуск висящих линий, чтобы отфильтровать те линии, длина которых будет больше допуска.
